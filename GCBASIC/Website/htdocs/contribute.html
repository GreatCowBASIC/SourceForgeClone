Do you have some spare time that you'd like to put to good use? Would you like to help out with the development of Great Cow BASIC? If so, we'd love to hear from you!

How can I help?
There are many ways in which help is welcome - some possible ideas:
  • Write some additional built-in routines
  • Translate GCBASIC into a language other than English
  • More documentation
  • Native Linux build
Who do I contact?
There are two ways in which you can get in touch:
  • GCBASIC Forums
  • Email the lead developer
