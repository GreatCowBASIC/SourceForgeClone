Great Cow BASIC Compiler

Two versions of GCBASIC are available. The older version, which is easier to install and has all of the correct translations in it, or the newer version, which has less bugs and more features, but doesn't have up to date translations or an installer. For AVR support, you will need the newer version.

Files marked in bold are the recommended downloads.

File Date Description
GCBASIC_Installer_0930.exe 24/2/2007 Installer for GCBASIC. Includes gputils, which this version requires to assemble programs.

Note: The version of gputils included in this file is now outdated

As the full installer of GCBASIC includes gputils.  Please see here for  updating  gputils. This link also provides information regarding the authors.
GCBASIC_Minimal_0930.exe 24/2/2007 Installer for GCBASIC. this installer does not include gputils. This is useful  if you already have MPASM or gputils installed, or you wish to download separately.  See here for the download of gputils.

Also use this version of the installer  if you are going to install but need the Italian or Turkish translations.
GCBASIC_zip_0930.zip 24/2/2007 Archive containing only GCBASIC. Extract to a folder to run.

update-nochipdata.zip
 22/9/2013 Archive containing the latest version of GCBASIC, GCBASIC with chip data files removed.

You can use this by itself, or extract it over an existing installation of GCBASIC to update it.

If this is the first time you have used the update version, you will also need to save gcbasic.ini to your installation folder. gcbasic.ini  is not included in any updates - including it could result in someone's customised settings being removed.

A file listing the changes made in each newer build is available here.
