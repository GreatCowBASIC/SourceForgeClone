InitPWM
 Previous  Next
Syntax:    
    InitPWM

Explanation:
This command will initialise the PWM module of the PIC chip, according to the settings specified by the configuration constants. InitPWM must be called at the start of the program if PWM is to be used.

Example:
    'This program will enable a 76 Khz PWM signal, with a duty cycle
    'of 80%. It will emit the signal for 10 seconds, then stop.
    #define PWM_Freq 76    'Set frequency in KHz
    #define PWM_Duty 80    'Set duty cycle to 80 %
    InitPWM            'Set up the PWM module
    PWMOn                'Turn on the PWM
    WAIT 10 s            'Wait 10 seconds
    PWMOff            'Turn off the PWM
