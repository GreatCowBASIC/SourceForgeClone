LCDWord
 Previous  Next
Syntax:    
    LCDWord value

Explanation:
The LCDWord command will show the specified value on the LCD, at the current cursor position. LCDWord can accept input from both 8 and 16 bit variables, allowing it to display values up to 65535.

LCDWord uses considerable more memory and takes longer to run than LCDInt. If the highest number to display is less than 255, please use LCDInt instead.

Example:
Please see the LCDInt example - the two commands have an identical syntax, and require the same constants to be set.
