The built-in #defines used to support  #IFDEF  etc are as follows:

    ChipMHz
    ChipName
    ChipFamily  this returns 12, 14 or 16, depending on instruction width
    OSC
    
The directives Var(), NoVar(), Bit() and NoBit() are functions that are built in to #IFDEF.  They will return true if a variable or bit is declared/not declared in the currently selected PIC chip.
    Var()
    NoVar()
    Bit()
NoBit()
The directive AllOf and OneOf will return true if all of or one of the listed defines is declared
    AllOf(define1, define2, …)
    OneOf(define1, define2, …)
