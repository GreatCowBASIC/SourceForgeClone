Introduction
The Great Cow Basic GLCD drivers support the GCB font set.  The default fonts that is embedded into the drivers supports ASCII characters 32-127.

GCB also supports and extended font set that supports ASCII characters 32-254.  The default ASCII set is GREEK.  To access the extended font set add the following to your main program.

        #define GLCD_EXTENDEDFONTSET1

The fonts can be resized by setting the global variable as follows:

GLCDfntDefaultsize = 1 | 2 | 3 | 4


Changing the extended font set
This extended font set can be replaced by an alternative fonts set you can add your own font tables by adding the following definitions.  Each table needs to 239 bytes in  length and are bitwise character definitions.

#define GLCDCharCol3 GLCDCharCol3MyFontDefintion
#define GLCDCharCol4 GLCDCharCol4MyFontDefintion
#define GLCDCharCol5 GLCDCharCol5MyFontDefintion
#define GLCDCharCol6 GLCDCharCol6MyFontDefintion

#define GLCDCharCol7 GLCDCharCol7MyFontDefintion

The table definitions in this example would be called GLCDCharCol[N]MyFontDefintion where [N] must be 3 to 5.
