Color Support
 Previous  Next
The Great Cow Basic GLCD drivers support color.  The following drivers support color.

1.    ILI9340
2.    SDD1289
3.    ST7735
These GLCD specific commands support color drivers as a parameter of color.  If a  GLCD specific command is not passed a color parameter then the current GLCDForeground color will be used.
