LCDInt
 Previous  Next
Syntax:    
    LCDInt value

Explanation:
The LCDInt command will show the specified value on the LCD, at the current cursor position.

LCDInt is limited to numbers between 0 and 255 - for larger numbers, please use LCDWord.

Example:
    'A Light Meter program.

    'General hardware configuration
    #chip 16F877A, 20
    #mem 368
    #define LightSensor AN0

    'LCD connection settings
    #define LCD_IO 8
    #define LCD_DATA_PORT PORTC
    #define LCD_RS PORTD.0
    #define LCD_RW PORTD.1
    #define LCD_Enable PORTD.2

    CLS
    PRINT "Light Meter:
    locate 1,2
    PRINT "A GCBASIC Demo"
    wait 2 s

    do while true
     CLS
     PRINT "Light Level: "
     LCDInt(ReadAD(LightSensor))
     wait 25 10ms
    loop
