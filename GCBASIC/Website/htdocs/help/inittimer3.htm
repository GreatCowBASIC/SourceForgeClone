InitTimer3
 Previous  Next
Syntax:
    InitTimer3 source, prescaler

Command Availability:
Available on all microcontrollers with a Timer 3 module.

Explanation:
InitTimer3 will set up timer 3, according to the settings given. source can be Osc , Ext or ExtOsc (ExtOsc is only available on PIC microcontrollers).
