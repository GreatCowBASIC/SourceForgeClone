InitTimer2
 Previous  Next
Syntax:
    InitTimer2 prescaler, postscaler  for PIC

    InitTimer2 source, prescaler for AVR

Command Availability:
Available on all microcontrollers with a Timer 2 module.

Explanation for PIC:
InitTimer2 will set up timer 2, according to the settings given.On a PIC.

A postscaler slows the rate of the interrupt generation(or WDT reset) from a counter/timer by dividing it down.  Postscaler values from 1:1 to 1:16.
