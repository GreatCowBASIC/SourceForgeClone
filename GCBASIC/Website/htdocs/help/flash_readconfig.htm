FLASH_readconfig
 Previous  Next
Syntax:
FLASH_readconfig(address)

Address is an absolute Flash memory address.

Command availability:
Available on specific Microchip MCUs only. Check your datasheet.

Explanation:
This function is a variant of the FLASH_read function that allows access to addresses above the 0x8000 threshold where the IDLOC information, configuration bits and calibration data are found.


Example:
#chip 16F1509, 8
Dim result as word
; the following statement will return in result
; the value of the Device ID of a PIC16F1709

result=Flash_Readconfig(0x8006)
