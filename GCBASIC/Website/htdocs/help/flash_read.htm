FLASH_read
 Previous  Next
Syntax:
FLASH_read(address)

Address is an absolute Flash memory address.

Command availability:
Available on specific Microchip MCUs only. Check your datasheet.

Explanation:
This function reads a 14-bit word from a given address in the device FLASH memory

Example:
#chip 16F1509, 8
Dim result as word
; the following statement will return in result
; the value of Memory location 0x1F80

result=Flash_Readconfig(0x1F80)
