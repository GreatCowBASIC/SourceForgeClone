InitTimer0
 Previous  Next
Syntax:
    InitTimer0 source, prescaler

Command Availability:
Available on all microcontrollers with a Timer 0 module.

Explanation:
InitTimer0 will set up timer 0, according to the settings given. source can be Osc or Ext.

Notes for Timer 0

On a Microchip Timer0 is a free running timer on midrange and
baseline microprocessors.  On these chips Timer0 always runs
and cannot be stopped.   However on most 18Fxxx microprocessors,
timer0 can be started or stopped via the T0CON.7 Bit.  Therefore, on these chips timer0 MUST be started with the following command:

StartTimer 0

On Many 18F microprocessors timer0 can be either an 8-Bit or 16-Bit
timer.  On these chips this is determined by TC0CON.6 (T08BIT). The timer defaults to 8-bit. To set the timer to 16-Bit mode the GCB source code must include the  following line.

#define TMR0_16BIT

As an example using 16 bit mode with the 1:256 prescaler allows for very long timer periods ( > 8 sec @ 8Mhz).

Examples:
When the timer overflows from 255 to 0, a Timer0Overflow interrupt will be generated. This can be used in conjunction with On Interrupt to run a section of code periodically.

Example #1 for PIC:
'''This program adjusts the power to a LED over 2.5 seconds, from 100% then to 0% power again over 2.5 seconds.


#chip 16F886,16

; ----- Define Hardware settings
  Dir Portb.0 out


; ----- Constants
  #define LED Portb.0


; ----- Main body of program commences here.
    'Call the initialisation routine
    InitLEDConrol

    Do
        'Increase power to full over 2.5 seconds
        For LEDPower = 0 to 100
            LEDlum = LEDPower
            Wait 25 ms
        Next
        'Hold LEDPower
        Wait 1 s
        'Decrease power to zero over 2.5 seconds
        For LEDPower = 100 to 0
            LEDlum = LEDPower
            Wait 25 ms
        Next
        'Hold LEDPower
        Wait 1 s
    Loop

; ----- Support methods.  Subroutines and Functions
  'Setup routine
  Sub InitLEDConrol
      'Clear variables
      LEDlum = 0
      PWMCounter = 0

      'Add a handler for the interrupt
      On Interrupt Timer0Overflow Call PWMHandler

      'Set up the timer
      InitTimer0 Osc, PS0_2
      'Timer 0 starts automatically on a PIC,
    'for a AVR see below.
  End Sub


  'This will be called when Timer 0 overflows
  Sub PWMHandler
      If LEDlum > PWMCounter Then
          Set LED On
      Else
          Set LED Off
      End If
      PWMCounter += 1
      If PWMCounter = 100 Then PWMCounter = 0
  End Sub

Example #2 for AVR:
'''This program adjusts the power to a LED over 2.5 seconds, from 100% then to 0% power again over 2.5 seconds.

; ----- Configuration

#chip mega328p, 16
#define LED PORTB.5
dir LED out

'Call the initialisation routine
InitLEDControl


'Main routine
  Do
      ;Increase LED to full over 2.5 seconds
      For LEDSpeed = 0 to 100
          Wait 25 ms
      Next
      ;Hold LEDPower
      Wait 1 s
      ;Decrease LED to zero over 2.5 seconds
      For LEDSpeed = 100 to 0 step - 1
          Wait 25 ms
      Next
      ;Hold LEDPower
      Wait 1 s
  Loop
  end



'Setup routine
Sub InitLEDControl
    set LED off
    'Clear variables
    LEDSpeed = 0
    PWMCounter = 0

'   Add the handler for the interrupt
    On Interrupt Timer2Overflow Call PWMHandler

    'Set up the timer
    InitTimer2 Osc, PS_2_1
    StartTimer 2

End Sub


'PWM sub
'This will be called when the Timer overflows
Sub PWMHandler

    If LEDSpeed > PWMCounter Then
        Set LED On
    Else
        Set LED Off
    End If
    PWMCounter += 1
    If PWMCounter = 100 Then PWMCounter = 0

End Sub
