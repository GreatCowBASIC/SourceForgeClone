Using connection mode 10
The LCD is controlled via I2C of a type 10 LCD 12C adapter. Set LCD_10 to 10 for the YwRobot LCD1602 IIC V1 or the Sainsmart LCD_PIC I2C adapter
To use mode 10 you must define the I2C ports as normal in your GCB code. Then, define the LCD type, set the I2C_address of the LCD adapter and the LCD speed, if required.   Then set the backlight control, if required.
