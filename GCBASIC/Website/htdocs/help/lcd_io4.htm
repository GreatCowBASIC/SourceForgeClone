Using connection mode 4:
To use connection mode 4 the R/W, RS, Enable control lines and the highest 4 data lines (DB4 through DB7) must be connected to the microprocessor. 


Relevant Constants:
Specific constants are used to control settings for the Liquid Crystal Display routines included with GCBASIC.  To set these constants the main program should specific constants to support the connection mode using #define.
Constants required for connection mode 4.

Constant Name
Controls
Default Value
LCD_IO
Must be 4
4
LCD_RS
Specifies the output pin that is connected to Register Select on the LCD.
Must be set
LCD_RW
Specifies the output pin that is connected to Read/Write on the LCD. The R/W pin can be disabled*.
Must be set (use LCD_NO_RW if R/W is disabled)
LCD_Enable
Specifies the output pin that is connected to Enable on the LCD.
Must be set
 
 
 
LCD_DB4
Output pin used to interface with bit 4 of the LCD data bus
Must be set
LCD_DB5
Output pin used to interface with bit 5 of the LCD data bus
Must be set
LCD_DB6
Output pin used to interface with bit 6 of the LCD data bus
Must be set
LCD_DB7
Output pin used to interface with bit 7 of the LCD data bus
Must be set
LCD_LAT
Drives the port with LATx support.   Resolves issues with faster Mhz and the Microchip read/write/modify feature. See example below.
Optional


*The R/W pin can be disabled by setting the LCD_NO_RW constant. If this is done, there is no need for the R/W line to be connected to the chip, and no need for the LCD_RW constant to be set. Ensure that the R/W line on the LCD is connected to ground if not used!
