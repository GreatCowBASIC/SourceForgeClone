GLCDRotate
 Previous  Next
Syntax:
ROTATE LANDSCOPE | PORTRAIT_REV | LANDSCAPE_REV  | PORTRAIT

Explanation:

Rotate the GLCD display to a relative position.  The options are:
        LANDSCAPE
        PORTRAIT_REV
        LANDSCAPE_REV
        PORTRAIT

The command will rotate the screen and set the following variables to appropriate values.

[device driver name]_GLCD_WIDTH
[device driver name]_GLCD_HEIGHT
The command is supported by the following defines.
#define LANDSCAPE 1
#define PORTRAIT_REV 2
#define LANDSCAPE_REV 3
#define PORTRAIT 4
