HEFReadByte
 Previous  Next
Syntax:
HEFreadByte(Flash_rel_addr, Flash_offset)

Flash_rel_addr is the HEFM relative location (i.e. the row number)
Flash_offset is the offset within the row (between 0 & HEFLASH_ROWSIZE-1)

Command availability:
Available on specific Microchip MCUs only. Check your datasheet.

Explanation:
The HEFreadByte function return a byte variable containing the value stored in the HEFM on chips that support this feature.
The parameters of this function are the row (block) number and the offset with that row.

Example:
#chip 16F1509, 8
; The following example stores a string of characters
; in row 2 of the HEFM of the 16F1509 then reads a byte from it
;
; Store a string in row 2
Dim Hello as String
Hello="Hello GCB World!"
HEFwriteBlock(2,Hello,len(Hello))
;
; Retrieve the value of block 2, offset 6 and print it
print HEFreadByte(2,6)
