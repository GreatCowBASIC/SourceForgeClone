HSerGetNum
 Previous  Next
Command Availability:
Available on all microcontrollers with a USART or UART module.

Microchip supports USART1 and 2.
    AVR supports USART 1,2,3 and 4.


HSerGetNum is implemented as a subroutine.

Syntax:
HSerGetNum myNum     ‘Gets a multi digit number  from USART 1 HSerGetNum myNum,1 ‘Get a multi digit number from USART 1
HSerGetNum myNum,2 ‘Get a multi digit number from USART 2

Variable type is word so the number range is 0 to 65535


Explanation:
This command will read a multi digit number followed by a CR from an external serial source using a  hardware serial module.  The command checks that only numbers are input disregarding other characters while waiting for the ending <CR>.  It can be used only as a subroutine.

Remember to  “#define USART_BLOCKING” or USART_TX_BLOCKING in your program.

Example:
'This program receives a number and CR from a PC terminal and sends it back on both usarts
#chip 18f26k22, 16

'Set the pin directions
#define USART_BAUD_RATE 9600
#define USART_BLOCKING
#define USART2_BAUD_RATE 9600
#define USART2_BLOCKING

'Init pins
#define SerInPort PORTc.7    'usart1 in
#define SerOutPort PORTc.6    'usart2 out
  'Set pin directions
  Dir SerOutPort Out
  Dir SerInPort In
  Dir PORTB.6 Out            'USART2 out
  Dir PORTB.7 In            'USArt2 in
  Dir PORTB.0 Out            'leds for testing
  Dir PORTB.1 Out            'leds for testing
  Wait 100 Ms

; ----- Variables
    Dim myNum as Word
; ----- Main body of program commences here.
    'Message after reset
    HSerPrint "18F26k22"
    HSerPrintCRLF

    'Main routine
    Do forever
        'wait for char from UART
        'HSerReceive InChar
        HSerGetNum myNum,2    'from usart 2
        HSerPrint myNum,1            ' send out usart 1
        HSerPrint myNum,2        'send out usart 2
        HSerPrintCRLF 1,2        'send one CRLF out usart 2
      HserPrintCRLF 1,1        ‘send one CRLF out usart 1
    loop
