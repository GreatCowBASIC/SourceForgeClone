Using LCD_SPEED:
To use the IO connection mode 0, a subroutine to support the LCD write operations must be provided.
The communication performance of a LCD display can be controlled via a #DEFINE.   This method allow the displays to be connected to the microcontroller and the timing to be optimised.


    #DEFINE LCD_SPEED  FAST

Define
Required Connections
LCD_SPEED
Options are:
FAST - The speed is approximately 20,000 CPS.
MEDIUM - The speed is approximately 15,000 CPS.
SLOW - The speed is approximately 10,000 CPS.
If LCD_SPEED is not defined, the speed defaults to SLOW
