Line
 Previous  Next
Syntax:
Line(LineX1,LineY1, LineX2, LineY2, Optional LineColour = 1)

Explanation:
Draws a line on a graphic LCD from X1, Y1 location to X2,Y2 location.
Example:
#include <glcd.h>

line 0,0,127,63
line 0,63,127,0
line 40,0,87,63
line 40,63,87,0
Constraints:
Versions of GCB prior to May 2014 only supports lines in the vertical or horizontal axis.

The following addition to a Great Cow Basic program will tell the compiler to use the old line drawing routines.

#define line oldline
