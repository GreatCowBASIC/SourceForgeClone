HIC2WaitMSSP
 Previous  Next
Syntax:    
HI2CWaitMSSP


Command Availability:
Only available for Microchip microcontrollers.

Explanation:
The method ensure the MSSP module has completed operations before returning.

Only in master and slave mode.

Note:
    
The methods sets the global byte variable HI2CWaitMSSPTimeout to 255 or True if the MSSP module has timeout during operations. HI2CWaitMSSPTimeout can tested for the status of the I2C bus.

This command is also available on microcomputers with a second hardware I2C port.
